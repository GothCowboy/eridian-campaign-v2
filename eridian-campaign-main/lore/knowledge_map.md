# Knowledge Map
ID: system_knowledge_map

---

## @meta

Este sistema define qué personajes conocen qué información.

Regla absoluta:
El narrador NO puede usar información que el personaje presente no conoce o no puede inferir.

El conocimiento se divide en:

- known → conocimiento confirmado  
- suspected → sospecha o intuición  
- unknown → no disponible

# Reglas de uso narrativo

1. Un personaje solo puede actuar con base en:
   - known
   - suspected (con incertidumbre)

2. Información en "unknown":
   - no puede ser usada
   - no puede influir decisiones

3. El narrador debe:
   - cruzar personajes en escena
   - filtrar conocimiento antes de responder

4. Si hay duda:
   - asumir desconocimiento
